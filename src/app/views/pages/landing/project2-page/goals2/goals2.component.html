PROJECT GOALS

Soil Moisture Threshold

Determine soil moisture threshold values and other parameters for landslide.

Landslide Movement Model Toolbox Software

Develop or adopt a suitable landslide movement model toolbox software.

Landslide Management Information System

Develop and testing of a landslide management information system.

Mobile Application for Landslide

Develop a mobile application for landslide warning and awareness, reporting and routing to evacuation areas or safe spots.

Early Warning System

Develop a Web and SMS based early warning system with DRRMO Operation Center Dashboard.
